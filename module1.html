Unit 1: Definition of Web Programming, Internet and WWW

Web Programming is the process of creating websites and web applications. The Internet is a worldwide network connecting computers. WWW (World Wide Web) is a collection of web pages accessible through the Internet.

Unit 2: Internet Services and Communication Protocols

Internet services include communication and information sharing. Protocols are rules that allow devices communicate online.

  • Email
  • Video Conferencing
  • File Transfer
  • Web Browsing
Protocol Function
HTTP Transfers Web Pages
HTTPS Secure Web Communication
FTP Transfers Files
SMTP Sends Emails

Unit 3: Web Application Architecture and Network Model

A web application consists of Client, Server and Database.

Component Purpose
Client User Browser
Server Processes Requests
Database Stores Information
